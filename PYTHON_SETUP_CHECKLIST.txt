TROUBLESHOOTING
================================================================================

If "py: command not found":
‚Üí Restart Command Prompt
‚Üí Reinstall Python 3.11 with "Add to PATH" checked

If MediaPipe import fails:
‚Üí Make sure venv is activated (see "(venv)" in prompt)
‚Üí Reinstall: pip uninstall mediapipe && pip install mediapipe==0.10.8

If activation fails:
‚Üí Try: venv\Scripts\activate.bat
‚Üí Or use PowerShell: venv\Scripts\Activate.ps1
